# Tehzeeb – Online Fashion Brand 🌸

[🌐 Live Demo](https://tehzeeb-clothing-brand.vercel.app/)

Tehzeeb is a responsive e-commerce website showcasing premium women’s fashion collections: Lawn, Pret, Formal, and Unstitched. This project emphasizes modern UI/UX and a guest checkout system, allowing users to shop seamlessly without creating an account.

Features ✅
👉🏻Responsive Navbar: Logo, Home, New Arrivals, Collections dropdown, Sale 🔥, Lookbook, Contact, Search, Wishlist (using LocalStorage), Cart.
👉🏻Home Page: Hero section, Categories, New Arrivals, Featured Collection, Lookbook, Testimonials, Footer.
👉🏻Collection Page: Filterable product grid with hover effects for a modern shopping experience.
👉🏻Product Page: Add to Cart 🛒 and Buy Now ⚡ button for instant checkout.
👉🏻Cart Page: Quantity updates, total price calculation, and “Proceed to Checkout.”
👉🏻Guest Checkout System: No login required; users fill basic info and place order directly.
👉🏻Order Success Page: Order confirmation with unique Order ID and option to continue shopping.
👉🏻Wishlist: Save items in LocalStorage for users without login.
👉🏻Search System: JavaScript-based search for quick product lookup.

Tech Stack 💻
Frontend: HTML, CSS, JavaScript, Bootstrap, React.js (if applicable)
Tools: VS Code, Git & GitHub, Canva (for designs)

Project Highlights 🌟
Guest Checkout: Seamless shopping without login friction.
Dynamic Product Pages: Engaging UI with hover effects and instant purchase flow.
Portfolio-Ready: Demonstrates practical front-end skills and e-commerce workflow implementation.
